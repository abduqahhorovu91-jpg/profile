# Luxe Portfolio

A premium, mobile-first React portfolio app with a social-inspired experience, dark futuristic visuals, Framer Motion transitions, Tailwind styling, Zustand state, and mock project data.

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- React Router
- Zustand
- Sonner

## Run

```bash
npm install
npm run dev
```

## Structure

- `src/components` reusable UI and layout
- `src/pages` route-level screens
- `src/data` mock portfolio content
- `src/store` saved state and chat state
